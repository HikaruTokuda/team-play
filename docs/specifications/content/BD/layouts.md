#### 出場成績確認画面  

自チーム分と相手チーム分の2セット用意。  

| # | コンポーネント名 | 説明 | コンポーネント種別 | 操作可否 |
| ---- | ---- | ---- | ---- | ---- |
| 1 | ヘッダ | ヘッダ項目を参照。<br>※ヘッダ・タイトルにはイベント名を設定。 | - | - |
| 2 | 出場成績（打撃） | 出場成績をリスト表示。<br>項目：<br>・打順<br>・選手名<br>・打席<br>・打数<br>・安打<br>・二塁打<br>・三塁打<br>・本塁打<br>・打点<br>・盗塁<br>・四球<br>・死球<br>・犠打<br>・犠飛<br>・打率<br>・OPS | テーブル | ✕ |
| 3 | 出場成績（投球） | 出場成績をリスト表示。<br>項目：<br>・選手名<br>・投球回数<br>・球数<br>・被安打<br>・奪三振<br>・失点<br>・自責点<br>・防御率<br>※勝利投手☆/敗戦投手★/ホールドH/セーブS | テーブル | ✕ |
| 4 | 出場成績編集ボタン | 出場成績登録画面へ遷移。<br>管理者のみ操作可能 | ボタン | 〇 |  

#### 出場成績登録画面  

自チーム分と相手チーム分の2セット用意。  

| # | コンポーネント名 | 説明 | コンポーネント種別 | 操作可否 |
| ---- | ---- | ---- | ---- | ---- |
| 1 | ヘッダ | ヘッダ項目を参照。<br>※ヘッダ・タイトルにはイベント名を設定。 | - | - |
| 2 | 出場成績（打撃） | 出場成績をリスト表示(編集可能)。<br>項目：<br>・打順<br>　打順/打/守備位置（漢字）/走から選択可能<br>・選手名<br>　所属選手から選択<br>　※助っ人登録用に自由記述も可<br>・打席<br>・打数<br>・安打<br>・二塁打<br>・三塁打<br>・本塁打<br>・打点<br>・盗塁<br>・四球<br>・死球<br>・犠打<br>・犠飛<br>・打率<br>　※入力不可。表示時に自動計算<br>・OPS<br>　※入力不可。表示時に自動計算 | テーブル | ✕ |
| 3 | 出場成績（投球） | 出場成績をリスト表示。<br>項目：<br>・選手名<br>・投球回数<br>・球数<br>・被安打<br>・奪三振<br>・失点<br>・自責点<br>・防御率<br>　入力不可。表示時に自動計算<br>・勝利/敗戦チェック<br>　勝利投手☆/敗戦投手★/ホールドH/セーブS | テーブル | ✕ |
| 4 | 確定ボタン | イベントに紐づく出場成績を確定。<br>出場成績確認画面へ遷移。 | ボタン | 〇 |

#### メンバ一覧画面
